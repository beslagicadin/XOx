# XOx

A desktop Tic Tac Toe game built with Visual C# and Windows Forms.

## Features

- Two-player local multiplayer (X vs O)
- 3x3 clickable game grid
- Win and draw detection
- New game button to reset the board

## Tech Stack

- Visual C#
- .NET Framework
- Windows Forms

## How to Play

1. Player X goes first
2. Click any empty cell to place your mark
3. First to get 3 in a row (across, down, or diagonally) wins
4. Click "New Game" to reset and play again

## Getting Started

1. Clone the repository
2. Open `progXO.sln` in Visual Studio
3. Build and run the solution

## Requirements

- Visual Studio
- .NET Framework
